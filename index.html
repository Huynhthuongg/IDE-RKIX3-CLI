AndroidX docs module

Giữ AndroidX như mẫu “library contribution workflow”.

Phần AndroidX vẫn hữu ích cho RKIX3 nếu đặt đúng vị trí: làm template docs cho các thư viện lớn cần checkout nặng, build artifact cục bộ, chạy test, rồi upload review. 

mkdir androidx-main
cd androidx-main
repo init -u https://android.googlesource.com/platform/manifest \
  -b androidx-main --partial-clone \
  --clone-filter=blob:limit=10M
repo sync -j8 -c
cd path/to/checkout/frameworks/support
ANDROIDX_PROJECTS=MAIN ./gradlew studio
./gradlew createArchive

AndroidX docs module

AndroidX được giữ như template docs cho thư viện lớn: checkout bằng repo, sync code, mở Android Studio, tạo Maven artifact cục bộ và upload review. 

mkdir androidx-main
cd androidx-main
repo init -u https://android.googlesource.com/platform/manifest \
  -b androidx-main --partial-clone \
  --clone-filter=blob:limit=10M
repo sync -j8 -c
cd frameworks/support
ANDROIDX_PROJECTS=MAIN ./gradlew studio
./gradlew createArchive
